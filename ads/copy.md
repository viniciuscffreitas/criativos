## Campaign Setup Checklist

- [ ] Objective: Engagement → Conversations (Click-to-WhatsApp)
- [ ] Optimization: Maximize Conversions
- [ ] Budget: €15/day, 1 campaign, 1-2 ad sets
- [ ] Placements: Instagram Feed, Stories, Marketplace
- [ ] Device: Desktop only
- [ ] Countries: Germany, Belgium, Portugal
- [ ] Age: 18-30
- [ ] Pre-filled WhatsApp message: "Hey, I saw your work — I need a website for my [business type]"
- [ ] Enable Advantage+ Creative (auto-generates format variations)
- [ ] Run 4-6 creatives simultaneously
- [ ] Rotate 2 new concepts every 2 weeks
- [ ] Kill any creative when frequency > 2.5/user/week
- [ ] Minimum 7 days before changing anything (learning phase)
