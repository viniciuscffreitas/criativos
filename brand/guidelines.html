01 — Logo

Logo system

The Vibe Web logo consists of the mark (VW symbol) and the wordmark "Vibe Web". Always use high-resolution versions from the original SVG files.

primary — dark background
for light backgrounds — vibeweb-black.svg
mark only — vibeweb-icon.svg
stacked — vibeweb-stacked.svg
← clear space = height of the letter "V" →
Do ✓
  • Use the green primary logo on dark backgrounds (#0a0a0a, #141414)
  • Use vibeweb-white.svg on any dark background
  • Use vibeweb-black.svg on light and white backgrounds
  • Always use high-resolution SVG source files
  • Minimum size: 120px wide (digital)
Don't ✗
  • Never use the green logo on a white background (low contrast)
  • Never distort, rotate, or alter the proportions
  • Never recreate the logo with different fonts
  • Never use the logo below the minimum size
  • Never apply shadows, outlines, or effects to the logo
